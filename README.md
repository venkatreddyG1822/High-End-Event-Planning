# STACKLY HTML Template
Premium Event Planning & Styling Studio (IMPERIA Events demo)

## Overview
STACKLY is a high-end, editorial-style HTML template for luxury event planning and styling studios. It features cinematic layouts, premium typography, dark/light mode, RTL support, and full client/admin dashboards.

## Quick Start
1. Open `pages/index.html` in your browser.
2. Keep the `assets/` folder alongside `pages/` so paths resolve correctly.
3. Use any static server if you prefer live reload (optional).

## Folder Structure
- `assets/css/style.css` (main styles + global system)
- `assets/css/darkmode.css` (dark mode overrides)
- `assets/css/rtl.css` (RTL overrides)
- `assets/js/main.js` (site interactions)
- `assets/js/dashboard.js` (dashboard interactions + charts)
- `pages/` (all HTML pages)
- `documentation/` (project docs)

## Customization Guide
### Branding
- Update the brand name/logo in the header of each page.
- Replace the icon (Font Awesome) or swap with your own SVG/logo.

### Colors
- Edit CSS variables in `assets/css/style.css` under `:root`.

### Typography
- Fonts used: Playfair Display (headings) + Manrope (body).
- Update the Google Fonts links in the HTML head if changing fonts.

### Images
- Replace background and hero images with your licensed assets.
- Use optimized formats (WebP preferred) for performance.

## Dark Mode
- Toggle by adding `dark-mode` to the `body` element.
- Dark mode overrides live in `assets/css/darkmode.css`.

## RTL Support
- Toggle by adding `rtl` to the `body` element.
- RTL overrides live in `assets/css/rtl.css`.
