# STACKLY HTML Template
## Overview for High-End Event Planning & Styling Studio

## Project Summary
This template presents a luxury event planning and styling studio with a premium, editorial feel. It showcases the planner's aesthetic through cinematic galleries and case studies while streamlining client communication via a full dashboard experience. The goal is a high-end, $10,000 ThemeForest-grade presentation with polished UX, accessibility, and dark/light + RTL support.

**Demo Brand Name:** IMPERIA Events (fully replaceable).

## Target Audience
Luxury wedding and destination clients, corporate brand experiences, and high-end private event clients seeking full-service planning and styling.

## Experience Goals
- Instantly communicate prestige and taste through large visuals, refined typography, and generous whitespace.
- Make the planning process feel effortless with clear steps, transparent pricing ranges, and guided CTAs.
- Provide a client portal that reduces back-and-forth (mood boards, vendor lists, contracts, payments).

## Information Architecture (Aligned to current `/pages`)
- Home Page 1 (`pages/index.html`): General services landing for premium events.
- Home Page 2 (`pages/homepage2.html`): Niche landing (luxury weddings and destination events).
- About Us (`pages/about.html`): Story, team, mission, history, awards, testimonials.
- Services (`pages/services.html`): Service grid + process overview.
- Pricing (`pages/pricing.html`): Package tiers + custom quote CTA.
- Contact (`pages/contact.html`): Form, map, studio info, social links.
- Login/Register (`pages/login.html`, `pages/register.html`).
- 404 (`pages/404page.html`).
- Coming Soon/Maintenance (`pages/comingsoon.html`).
- Admin Dashboard (`pages/admindashboard.html`): Studio operations and analytics.
- User Dashboard (`pages/userdashboard.html`): Client portal.

## Optional Pages (Not included yet)
- Service Details (`pages/service-details.html`): Deep dive into one service, inclusions, FAQs, pricing table.
- Blog (`pages/blog.html`): Editorial list with filters and search.
- Blog Details (`pages/blog-details.html`): Full article with sidebar.
- Portfolio/Gallery (`pages/portfolio.html`): Full gallery case studies.

## Signature Features
- Cinematic portfolio galleries with category filters (weddings, corporate, private, brand).
- Mood board gallery for each client with pinned items and comments.
- Curated vendor list with approval workflows and notes.
- Digital contracts and e-sign flow (placeholder integration).
- Pricing structure with transparent ranges and "custom experience" CTA.
- Event timeline and milestone tracker.

## Dashboard Modules
### Admin Dashboard
- Overview and analytics (lead sources, booking rate, revenue).
- Client pipeline and status tracking.
- Vendor management and preferred partners.
- Contracts and invoicing.
- Messages and notifications.
- Content management for case studies and blog posts.

### Client Dashboard
- Profile and event overview.
- Mood boards and inspiration collections.
- Vendor shortlist and recommendations.
- Proposals, contracts, and e-sign.
- Invoices, deposits, and payment status.
- Messaging and timeline milestones.

## Design Direction
- Visual Language: Editorial luxury, high-contrast photography, soft gradients, refined motion.
- Typography: One display serif + one clean sans-serif for body.
- Palette: Neutral base (ivory, stone) with metallic accents (champagne gold) and deep charcoal.
- UI Components: Elegant cards, pill tags, ghost buttons, rich hover states.

## Required Integrations (Placeholders)
- Contact form (Formspree/Netlify).
- Newsletter signup (Mailchimp/ConvertKit).
- Map placeholder (Google Maps).
- Calendar booking touchpoints.
- Payment buttons (Stripe/PayPal).
